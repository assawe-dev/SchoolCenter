## Project Overview
School Center Management System with user management, students records, and financial balance (Debit/Credit) tracking.

## Tech Stack & Constraints
- IDE: Visual Studio 2015
- Language: C# 6.0 (Strictly avoid newer C# features like records, pattern matching, or top-level statements)
- Framework: .NET Framework 4.6
- Database: SQL Server
- Data Access: ADO.NET (SqlConnection, SqlCommand)
- UI: Windows Forms (WinForms)
